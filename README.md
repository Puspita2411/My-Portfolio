# Personal Developer Portfolio 🚀

A premium, highly interactive developer portfolio featuring a glassmorphic dark-theme design, custom micro-animations, scroll-triggered reveals, and a serverless form delivery integration.

[![Deploy to Render](https://render.com/images/deploy-to-render-button.svg)](https://render.com/deploy?repo=https://github.com/Puspita2411/My-Portfolio)

---

## 🎨 Key Features
- **Modern Design**: Dark-mode glassmorphic user interface styled with high-fidelity custom CSS.
- **Micro-Animations**: Custom scroll-triggered fades, dynamic floating badges, and typed role greetings.
- **Particle System**: Lightweight, pure-JS canvas particle background in the hero section.
- **Serverless Form**: Automated contact forms integrated with Web3Forms to deliver inquiries directly to Gmail.
- **Project Grid**: Structured showcase sections highlighting tools, category tags, and solved challenges.

---

## 🛠️ Tech Stack
- **Structure**: HTML5 (Semantic elements)
- **Styling**: CSS3 (Custom properties, CSS Grid, Flexbox, blur backdrops)
- **Logic**: Modern JavaScript (Intersection Observer API, dynamic filters, localStorage theme state)
- **Form Delivery**: Web3Forms API
- **Deployment**: Render (via `render.yaml` static hosting)

## 🚀 How to Deploy on Render
Click the button at the top of this file, or follow these steps:
1. Log in to [dashboard.render.com](https://dashboard.render.com).
2. Click **New +** and select **Blueprint**.
3. Connect your **`My-Portfolio`** repository.
4. Render will automatically read the `render.yaml` configuration and launch your site!
